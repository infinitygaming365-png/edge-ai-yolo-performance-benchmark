# Edge AI Performance Benchmark using YOLOv8 on NVIDIA T4

## Objective
Evaluate performance of YOLOv8 models under different architectures and resolutions.

## Hardware
- NVIDIA T4 GPU
- Google Colab

## Experiments
- Model comparison (YOLOv8n, YOLOv8s, YOLOv8m)
- Resolution comparison
- FPS benchmarking

## Results
- YOLOv8n fastest
- YOLOv8m more detailed but slower
- 640 resolution best performance balance

## Conclusion
Edge AI requires balancing speed and accuracy depending on deployment environment.
